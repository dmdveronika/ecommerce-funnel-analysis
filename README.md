# E-commerce Conversion Funnel Analysis
This project analyzes user behavior in an e-commerce website using GA4 event data.

## Goal
The goal of this project was to analyze the conversion funnel from session start to purchase and identify where users drop off.

## Dataset
GA4 public e-commerce dataset from Google BigQuery.

## Tools
SQL (Google BigQuery)
Tableau Public

## Steps
1. Data extraction from GA4 event tables
2. Creating a unique session identifier
3. Building a conversion funnel
4. Calculating conversion rates
5. Creating an interactive Tableau dashboard

## Key Metrics
Total Sessions: 354,856  
Total Purchases: 4,745  
Conversion Rate: 1.34%

## Key Insight
The largest drop-off occurs between session start and product view, indicating potential issues with landing pages or traffic quality.
